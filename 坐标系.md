在惯性导航（IMU）姿态解算中，理解不同的坐标系及其转换关系是准确描述运动的基础。以下是惯导中最常用的四个坐标系及其定义、用途和相互转换方法。



### 1. 惯性坐标系（i 系）

- **定义**：原点位于地心，坐标轴指向遥远恒星（不随地球自转）。在工程近似中，通常采用**地心惯性系**（ECI，Earth-Centered Inertial）。
- **轴方向**：X 轴指向春分点，Z 轴沿地球自转轴指向北极，Y 轴按右手定则确定。
- **用途**：牛顿力学定律（$F = ma$）严格成立的坐标系。IMU 中的陀螺仪直接测量相对于惯性系的角速度，加速度计测量的是相对于惯性系的比力（非引力加速度）。它是所有导航计算的终极参考。





### 不同坐标系在 IMU 算法中的角色

| 坐标系   | 算法中的角色                                                 |
| :------- | :----------------------------------------------------------- |
| **i 系** | 陀螺仪积分的基础，但一般不直接使用，而是通过补偿地球自转转换为相对 n 系的角速度。 |
| **e 系** | 用于 GPS/INS 组合导航，将 GPS 的经纬度与惯导推算的位置对齐。 |
| **n 系** | 姿态解算的参考系，导航方程（速度、位置）的主要积分坐标系。   |
| **b 系** | 传感器原始数据的坐标系，所有滤波（互补滤波、卡尔曼滤波）的观测值来源。 |
